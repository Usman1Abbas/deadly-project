# Contributing to deadly project

We welcome contributions to deadly project! Please follow these guidelines:

1.  Fork the repository.
2.  Create a new branch for your feature or bug fix.
3.  Make your changes and commit them with descriptive commit messages.
4.  Submit a pull request to the `main` branch.

## Code Style

Please follow the existing code style in the project.

## Testing

Please write unit tests and integration tests for your code.

## Documentation

Please update the documentation if you add or modify any features.
